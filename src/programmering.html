Variabler

Flere ting samtidig

navn = "Emma"
alder = 14
print("Hei, jeg heter", navn, "og er", alder, "år gammel.")

Favorittfarge

navn = "Emma"
favorittfarge = "blå"
print("Favorittfargen til", navn, "er", favorittfarge)

Brukerinput – navn

Brukerinput – tall

tall = float(input("Skriv inn et tall: "))

print("Tallet delt på 2 er", tall / 2)
print("Tallet opphøyd i 4 er", tall ** 4)

Vanlige datatyper i Python

Datatype Eksempel Beskrivelse
int 5, -3 Heltall
float 3.14, -0.5 Desimaltall
str "tekst", 'abc' Tekst (streng)
bool True, False Sannhetsverdi
list [1, 2, 3] Liste med flere verdier
dict {"navn": "Anna"} Ordbok med nøkkel-verdi-par
tuple (4, 5) Uforanderlig liste
set {1, 2, 3} Mengde med unike verdier
NoneType None Mangler verdi (null-verdi)
