Sirkel

Radius: r
Omkrets: O = 2 · π · r
Areal: A = π · r2

Rektangel

Omkrets: O = 2 · (l + b)   /   l + n + l + b
Areal: A = l · b

Parallellogram

Omkrets: O = 2 · (g + s)   /   g + s + g + s
Areal: A = g · h

Trekant

Omkrets: O = a + b + c
Areal: A = (g · h) / 2

Likesidet trekant

Omkrets: O = 3 · a
Areal: A = (a² · √3) / 4

Likebeinet trekant

Omkrets: O = 2 · s + b
Areal: A = (b · h) / 2

Rettvinklet trekant

Omkrets: O = a + b + c
Areal: A = (a · b) / 2
Pytagoras: c² = a² + b²

Huskeregler for trekanter

  • I en rettvinklet trekant med vinkler 30° og 60°:
    • Hypotenusen er dobbelt så lang som den korteste kateten.
    • Den lengste kateten er √3 ganger den korteste.
  • I en rettvinklet trekant med vinkler 45° og 45°:
    • Katetene er like lange.
    • Hypotenusen er lik katet · √2.
  • Summen av vinklene i en trekant er alltid 180°.
  • I en likesidet trekant er alle vinkler 60° og alle sider like lange.
