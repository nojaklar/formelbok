Geometriske Figurer og Formler

Sirkel

Radius: r
Omkrets: O = 2 · π · r
Areal: A = π · r²

Rektangel

Omkrets: O = 2 · (l + b)   /   l + n + l + b
Areal: A = l · b

Parallellogram

Omkrets: O = 2 · (g + s)   /   g + s + g + s
Areal: A = g · h

Trekant

Omkrets: O = a + b + c
Areal: A = (g · h) / 2

Trapes

Omkrets: O = a + b + c + d
Areal: A = ((a + b) / 2) · h

Prisme (rektangulært)

Volum: V = l · b · h

Sylinder

Volum: V = π · r² · h
Areal av overflate: A = 2 · π · r² + 2 · π · r · h

Kule

Volum: V = (4/3) · π · r³
Areal: A = 4 · π · r²
